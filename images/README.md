# Images Directory

Place your screenshot images here:

- `insights-screenshot.png` - Category profit breakdown screenshot (used in hero and feature sections)
- `pricing-calculator.png` - Pricing calculator screenshot
- `product-list.png` - Product tracking/dashboard screenshot
- `mobile-dashboard.png` - Mobile app screenshot (optional, 400px width recommended)

## Image Requirements

- Recommended format: PNG or WebP
- Recommended sizes:
  - `insights-screenshot.png`: 900px width (max-width in HTML)
  - `pricing-calculator.png`: Full width of container
  - `product-list.png`: Full width of container
  - `mobile-dashboard.png`: 400px width
- Optimize images for web (use tools like TinyPNG or ImageOptim)

The HTML will display beautiful placeholder boxes with gradients if images are missing, so you can test the layout before adding actual screenshots.
